# SWAT Bot

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ccostello102711/SWAT-Bot.git
   cd SWAT-Bot
   ```

2. **Install dependencies**:
   If using Node.js, run:
   ```bash
   npm install
   ```

3. **Configure environment variables**:
   Create a `.env` file in the root directory and add your configurations as follows:
   ```
   TOKEN=your_bot_token_here
   OTHER_CONFIG=your_value_here
   ```

4. **Run the bot**:
   ```bash
   npm start
   ```
